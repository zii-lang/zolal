# Step 0: Usage
After haveing a compiled version of `zolal` you can use it in command-line:

```bash
zolal --version
```
```
zolal version 1.0.0

zolal is library manager and preprocessor for Z.
```

## build & source directory
You can specify the location of `manifest.zl` and where source is using:

```bash
zolal -S <source-dir>
```

You can specify the location of build:

```bash
zolal -B <build-dir>
```
